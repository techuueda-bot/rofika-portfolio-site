# RO FIKA Detail Copy

## Overview

RO FIKAは、架空の小さな夏のカフェを題材にした1ページサイトです。
冷たい飲みものと焼き菓子をそばに、窓辺で静かにひと息つける時間をWeb上で表現しました。

## Concept

価値として置いたのは、「ひとりでも静かに過ごせそうな、光の入るカフェ時間」です。
大きな情報量で説明するのではなく、短い日本語コピー、窓辺の写真、淡い色、余白で空気感を作っています。

## Design

HeroとConceptでは、窓辺の光を感じる写真と小窓モチーフを使い、RO FIKAの第一印象を作りました。
Menuは日本語を主役にし、商品名・英語名・説明文を読みやすく整理しています。
Galleryには実写真を入れ、架空サイトでありながらカフェとしての実在感が伝わるようにしました。

## Sections

- Hero: RO FIKAの名前、夏のカフェ感、窓辺の光を伝える導入
- Concept: 「光の入る窓辺で、ひと息。」という世界観を短く説明
- Menu: 冷たい飲みものと焼き菓子を、価格帯つきで自然に掲載
- FIKA TIME: カフェで過ごす静かな時間を言葉と余白で表現
- Gallery: 窓辺、グラス、焼き菓子、午後の光を実写真で補足
- Footer Info: Open / Close / Place / Seats / Payment / Instagramを掲載

## Marketing Adjustment

制作後半では、見た目だけでなく「カフェサイトとして安心して読めるか」を見直しました。
Menuには価格帯を追加し、Footer Infoには営業時間、定休日、場所、席数、支払い、Instagramを整理して掲載しています。
また、Conceptに「ひとりで過ごす午後にも。」を加え、ひとり利用のしやすさが伝わるようにしました。

## Implementation

HTML / CSS / JavaScriptで制作した、静的な1ページサイトです。
一部要素には `data-drift` による控えめなスクロール演出を加えています。
一方でGalleryには `data-drift` を付けず、写真の実在感と表示の安定性を優先しました。

## Responsive

スマホ表示では、文字量が重くならないよう余白とサイズを調整しています。
Galleryは2列、Footer Infoは1列表示にし、横スクロールが出ないようにしています。

## Learning

この作品では、雰囲気づくりと実用情報のバランスを学びました。
余白や世界観を守りながらも、価格帯や営業時間などの情報を少し加えることで、「きれいな作品」から「カフェとして読めるWebサイト」へ近づけています。
