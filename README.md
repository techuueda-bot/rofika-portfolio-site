# RO FIKA

## Live Site

https://techuueda-bot.github.io/rofika-portfolio-site/

## 概要

RO FIKAは、架空カフェを題材にした1ページ構成のブランドサイトです。
夏の午後に少し立ち止まるような、軽くて余白のあるカフェ体験をWeb上で表現することを目的に制作しました。

## コンセプト

A small pause in a summer afternoon.

夏の午後、
冷たいドリンクと焼き菓子をそばに置いて、
少しだけ呼吸をゆるめるためのFIKA。

## 制作目的

この作品では、単に店舗情報やメニューを並べるのではなく、
写真、余白、英字タイポグラフィ、小窓モチーフ、軽いスクロール演出を組み合わせて、
架空ブランドの空気感を設計することを目的にしました。

## ターゲット

・カフェや暮らしの空気感が好きな人
・余白のあるWebデザインが好きな人
・北欧、焼き菓子、器、窓辺の光などに関心がある層
・ポートフォリオ閲覧者に対して、明るく軽いブランドサイト表現を見せる目的

## 参考にした方向性

RO FIKAでは、参考サイトの見た目をそのまま真似るのではなく、役割を抽出して再構成しました。

### せきがはら人間村

借りた役割：
余白、自然、短い言葉で世界観を作る力。

RO FIKAでは、長い説明ではなく、短いコピーと余白で夏の午後の空気を作る方向に活かしました。

### BLUE NOTE PLACE

借りた役割：
英字タイポグラフィ、飲食を体験として見せる設計、時間帯やメニューの整理。

RO FIKAでは、英字の強弱やMenu / FIKA Timeの構成に活かしました。
ただし、音楽的・都会的な躍動感には寄せすぎず、静かな夏のカフェ体験に変換しています。

### Chevron Coffee

借りた役割：
写真、文字、余白がなめらかにつながるスクロール感。

RO FIKAでは、スクロール時に要素が少し遅れて現れる軽い動きとして活かしました。
見た目やレイアウトの直接的な模倣は避けています。

## セクション構成

1. Hero
2. Concept + Window Scene
3. Menu
4. FIKA Time
5. Gallery
6. Footer

## デザイン上の工夫

・ミルクホワイト、淡いブルー、薄いレモン、ベージュを中心にした配色
・小窓モチーフによる「窓辺の光」の表現
・英字タイポグラフィの強弱
・Menuを商品一覧ではなく、カフェ体験を補強する編集面として設計
・FIKA Timeで「飲む、食べる、眺める、少し休む」という時間の流れを表現
・Galleryは写真差し替え前提の軽い骨組みとして設計

## 実装上の工夫

・1ページ完結の縦スクロール構成
・CSS中心の軽いビジュアル表現
・重たいライブラリは追加しない
・スクロール演出は一部要素に限定
・Galleryは安定性を優先してスクロール演出対象から外す
・レスポンシブ対応として、スマホでは文字サイズや余白、Galleryの2列表示を調整

## アクセシビリティ・品質確認

・h1はHeroに1つ
・各セクションはh2で整理
・Menu内はh3を使用
・Skip linkあり
・Galleryは後から画像altへ移せる data-alt 構造あり
・横スクロールなし
・Footer最下部までスクロール確認済み
・コンソールエラーなし
・Chromeクラッシュなし
・Gallery内 data-drift は0

## 今後の改善案

・Galleryの写真差し替え後、余白と色味を再調整する
